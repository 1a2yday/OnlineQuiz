# 英语刷题助手 🍬

糖果风手机适配答题应用，支持三种题型，纯前端静态页面，可部署到 GitHub Pages。

## 题型支持

| 题型 | 说明 |
|------|------|
| **单选题** | 四选一，点击确认 |
| **消消乐** | 左列英文 → 右列中文配对，右侧每次随机打乱 |
| **还原句子** | 按正确顺序点击单词排列 |

- 所有题型支持答案解析（`explanation` 字段）
- 作答流程：选择 → 确认 → 显示对错+解析 → 手动点下一题
- 错题自动收集，支持错题重刷模式

## 主题系统

4套糖果色主题，主页底部一键切换：
- 🍯 **蜂蜜柠檬糖**（默认暖橘色 `#FF8C42`）
- 🍃 薄荷糖 `#00BFA5`
- 🍓 草莓糖 `#FF6B9D`
- 🫐 蓝莓糖 `#667EEA`

## 快速开始

```bash
# 安装依赖（pnpm 不可用时用 npm --ignore-scripts）
npm install --ignore-scripts

# 启动开发服务器
npm run dev:local

# 构建生产版本
npm run build:local

# 预览构建产物
npm run preview
```

## 题库生成工具

`tools/quiz-generator.html` — 双击即用，无需服务器。

在 Excel 中按格式填写，复制粘贴即可生成题库 JSON：

| A列(题型) | B列 | C列 | D列 | E列 | F列 | G列 | H列 |
|-----------|-----|-----|-----|-----|-----|-----|-----|
| 单选 | 题目 | 选项A | 选项B | 选项C | 选项D | 答案(0-3) | 解析(可选) |
| 消消乐 | 说明 | 左词\|.. | 右词\|.. | 配对(可选) | 解析(可选) | — | — |
| 还原句子 | 中文提示 | 单词\|.. | 正确顺序(可选) | 解析(可选) | — | — | — |

> **消消乐**：E 列（配对）可选，留空自动按左右顺序一一配对。答题时右侧每次随机打乱。
> **还原句子**：D 列（顺序）可选，留空自动按单词书写顺序。答题时单词随机打乱显示。

## 静态部署

`dist/` 目录可直接上传到 GitHub Pages 等任意静态托管服务，无需后端。

### 自定义内容

| 修改内容 | 编辑文件 | 说明 |
|----------|---------|------|
| 横幅图 | 替换根目录 `banner.gif` | 部署到 `dist/` |
| 欢迎文字 | `src/storage.ts` | 搜索 `welcomeTitle` / `welcomeSubtitle` |
| 默认主题 | `src/themes.ts` | 修改 `DEFAULT_THEME_ID` |
| 管理员密码 | `src/storage.ts` | 搜索 `adminPassword` |

修改后需重新 `npm run build:local`。

## 技术栈

- Vite 7 + TypeScript
- Tailwind CSS 3（自定义 candy 色板）
- CSS 变量驱动的多主题系统
- Hash SPA 路由，localStorage 数据存储
