# DESIGN.md

## 气质与意象
- **意象锚点**：糖果店里的英语学习角——透明玻璃罐里装满彩色波板糖，阳光透过糖纸在桌面上投下斑斓光影。甜而不腻，活泼但不幼稚。
- **关键词**：甜美、轻快、糖果罐、波板糖、彩虹糖粒

## 配色方案（4套主题，用户可切换）

### 默认主题：蜂蜜柠檬糖 🍯
- 主色：`#FF8C42` — 按钮、强调元素、标题
- 辅色：`#FFB347` — 次要按钮、标签
- 点缀色（薄荷糖）：`#2EC4A7` — 正确状态、成功反馈
- 暖色（柠檬糖）：`#FFC940` — 星星、徽章、高亮
- 背景（奶油）：`#FFF8F0` — 页面底色

### 可选主题
- 草莓糖 🍓：`#FF6B9D` / 薄荷糖 🍃：`#00BFA5` / 蓝莓糖 🫐：`#667EEA`

### 通用规则
- 卡片：`#FFFFFF` 带暖调阴影
- 文字主色：深紫灰/深棕，柔和但不刺眼
- 所有颜色通过 CSS 变量驱动，切换主题即时生效

## 字体排版
- 中文：`Noto Sans SC`（Google Fonts，通过 fonts.googleapis.cn 引入）
- 英文/数字：`Nunito`（圆润亲和，Google Fonts）
- 标题：加粗、圆角感
- 按钮文字：`Nunito` bold，略带 letter-spacing

## 动效与交互
- 运动性格：轻盈、有弹性，像糖果在桌面上弹跳
- 缓动：`cubic-bezier(0.34, 1.56, 0.64, 1)` 弹性缓出
- 按钮悬停：轻微上浮 + 阴影加深 + 0.2s 过渡
- 卡片进入：从下方淡入，带轻微弹性
- 正确反馈：薄荷糖色闪烁 + 小弹跳
- 错误反馈：轻微抖动（shake）

## 页面结构
- 手机适配（max-width: 480px 居中，超过则展示手机外壳）
- 顶部：自定义横幅图区域（支持 GIF）
- 中部：功能按钮区（圆角大按钮，糖果色）
- 底部：用户信息栏

## 设计禁忌
- 不要使用暗色模式
- 不要使用直角边框（所有元素至少 12px 圆角）
- 不要使用过于复杂的渐变色（最多双色渐变）
- 不要使用阴影过深（只用柔和的粉调阴影）
- 避免文字颜色过黑（使用深紫灰而非纯黑）
