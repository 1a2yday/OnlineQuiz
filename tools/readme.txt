  🛠️ 功能一览

  ┌───────────┬──────────────────────────────────────────────────┐
  │   功能    │                     操作方式                     │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 打开题库  │ 点「📂 打开 JSON」选择文件，或 Ctrl+O            │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 保存题库  │ 点「💾 下载 JSON」或 Ctrl+S                      │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 新建题库  │ 直接点左侧「+ 在开头插入」，选择题型             │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 编辑题目  │ 点击左侧列表中的题目，右侧显示对应表单           │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 插入题目  │ 点任意「+ 在此之后插入」按钮，选择题型           │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 删除题目  │ 点击题目旁的 ✕ 按钮，或编辑器底部「🗑️ 删除此题」 │
  ├───────────┼──────────────────────────────────────────────────┤
  │ 验证      │ 点「✅  验证」检查所有题目格式                    │
  ├───────────┼──────────────────────────────────────────────────┤
  │ JSON 预览 │ 底部折叠面板，实时更新，支持复制                 │
  └───────────┴──────────────────────────────────────────────────┘

  三种题型编辑器

  - 单选题 — 4 个选项输入框 + 下拉选择答案
  - 消消乐 — 配对表格，可添加/删除行，左右列对应编辑
  - 还原句子 — | 分隔单词输入，顺序自动推导

  使用方式

  双击 tools/quiz-editor.html 即可在浏览器中打开，无需服务器。编辑完成后下载 JSON，放回 public/quizzes/ 目录，push
  即可部署。
